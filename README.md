# App Android - Integração API + Banco de Dados

## Autor
Lucas Lima

## Descrição
Aplicativo Android desenvolvido em Kotlin com Jetpack Compose, integrando dados da Fake API e persistência local via Room.

## Tecnologias
- Kotlin
- Jetpack Compose
- Retrofit
- Room Database
- Coroutines
- MVVM

## Funcionalidades
- Consome dados da rota `/users` da Fake API
- Salva dados localmente no banco (Room)
- Exibe lista de usuários em tela (Compose)
